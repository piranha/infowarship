If you have a website, you can support Ukraine right now!

Not all Russian citizens have the information about what their armed forces are doing in Ukraine, and what casualties they've already experienced.

Add this script to your website to show a popup for visitors with Russian IP addresses with links to truthful news sources and information about prisoners of war.

By changing HTML code

In your site template before enclosing </body> or </head> tag add this code:

<script src="https://infowarship.com/script.js"></script>

With GTM

If you are using GTM you can add this popup without code change. Create a new tag of type "Custom HTML" and add this code:

<script src="https://infowarship.com/script.js"></script>

That's all
